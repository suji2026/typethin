# typethin

使用手机作为台式机语音输入源。手机说话 → 台式机实时输入文字。

## 使用方式

1. 台式机启动应用：`cd desktop && npm start`
2. 弹出二维码窗口，手机扫码
3. 手机浏览器打开页面，点击输入框
4. 切换到语音输入法（如豆包）开始说话
5. 文字自动出现在台式机光标所在位置

## 连接方式

手机和台式机只要网络可达即可，支持两种方式：

### 方式一：同一 WiFi（无线）

手机和台式机连同一个 WiFi 路由器。这是最简单的场景。

### 方式二：USB 数据线（台式机无 WiFi 时推荐）

台式机没有无线网卡时，用数据线连接手机：

1. iPhone 用数据线插入台式机 USB 口
2. 手机进入 **设置 → 个人热点**，开启 **USB 网络共享**
3. 手机会在台式机上创建一个虚拟网卡（通常在 `192.168.42.x` 段）
4. 启动本应用，程序会自动发现这个 IP 并生成对应二维码
5. 手机扫码即可

代码无需任何修改，`getLocalIP()` 会自动检测所有网卡。

## 要求

- Node.js >= 18
- 手机和台式机网络可达（WiFi 或 USB 线）
- Windows 10+（macOS 下键盘模拟需辅助功能权限）
- iPhone：开启个人热点中的 USB 网络共享

## 初始化

```bash
cd desktop
npm install
npm start
```
